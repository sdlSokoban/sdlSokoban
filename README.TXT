    sdlSokoban - A Sokoban game clone written in C++ and SDL

Author: Ericson Norberto
Email.: sdlsokoban@gmail.com

History

Sokoban (Sokoban, warehouse keeper) is a type of transport puzzle, in which the player
pushes boxes or crates around in a warehouse, trying to get them to storage locations.
The puzzle is usually implemented as a video game.
Sokoban was created in 1981 by  Hiroyuki Imabayashi, and published in 1982 by Thinking
Rabbit, a software house based in Takarazuka, Japan.
(http://en.wikipedia.org/wiki/Sokoban)

Rules

    1. Only one box can be pushed at a time.
    2. A box cannot be pulled.
    3. The player cannot walk through boxes or walls.
    4. The puzzle is solved when all boxes are located at storage locations.

Features
    
	1. .xsb file format compatible
	2. Map editor
	3. Support multiple full-screen resolutions
